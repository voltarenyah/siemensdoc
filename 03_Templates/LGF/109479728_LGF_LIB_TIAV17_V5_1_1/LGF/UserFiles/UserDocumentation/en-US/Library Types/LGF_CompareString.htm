LGF_CompareString (FC / V1.0.0)

Overview
Author: Siemens Industry Support

Short description

Compares two strings and returns a number which indicates the result of the comparison.

Block Interface

LGF_CompareString (FC)
   Ret_Val  SInt
    
String  left  String
       
String  right  String
       
   

Output parameter

IdentifierData type Description
Ret_ValSInt Return values:
left < right := -1;
left > right := 1;
left == right := 0;

In/Out parameter

IdentifierData type Description
leftString Left / first string to compare
rightString Right / second string to compare

Code / ValueIdentifier / Description
